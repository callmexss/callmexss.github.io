测试标准输出

有些时候希望测试标准输出是否符合某种期望，可以使用上下文管理器（context manager）捕获标准输出。这种方法的本质也是重定向标准输出，使用上下文管理器的好处是不需要在测试结束后还原标准输出，可以避免使用 try...finally 块。
