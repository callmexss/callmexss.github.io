Eclipse快捷键

发表于 2017-07-11 | 更新于 2019-04-01   |   93 字   |   1 min | 分类于 tools | 评论数： | 阅读次数：

目前感觉最常用的有：

alt + / 自动补全
ctrl + / 注释（取消注释）当前选定的内容
ctrl + d 删除当前行
ctrl + m 最大化（还原）当前窗口
ctrl + e 打开编辑器（切换窗口）
ctrl + o 显示大纲
ctrl + shift + f 格式化代码
ctrl + shift + o 导入及管理import语句
ctrl + shift + t 打开Open Type 查找类文件
